# Prey

The **Prey** is a land-favoring herbivore species within the animals ecosystem. They serve as the primary food source for the predator species (Snakes) while simultaneously striving to graze, survive, and multiply.

## Behavior Profile

- **Role**: Land-favoring Herbivore
- **Diet**: Grass. Grazing depletes the cell's grass health, which slowly regrows over time.
- **Movement**: Highly capable on solid terrain (Grass, Sand) but significantly slowed down by Water.
- **Goal**: Evade predators, seek out fresh grass to graze on, and eat enough grass to reproduce.

## AI & Observation Model

The Prey is driven by a custom PPO (Proximal Policy Optimization) model. 

### Observation Space (132 floats)

The Prey's observation vector (`PREY_OBS_SIZE = 132`) is generated by `get_prey_observation` and is identical in structure to the Amphibia's observation. 

1. **Entity & Terrain Grid (64 floats)**: 
   - An 8x8 grid in an absolute, non-rotating frame (up is always north).
   - Impassable obstacles (rocks, out-of-bounds) read as `-1.0`.
   - Snake heads (lethal) read as `-0.8`, and snake bodies read as `-0.5`.
   - Passable terrain is scaled by `Species::Prey.speed_on(terrain) * 0.5`. This causes water cells to read very low (~0.1), teaching the AI that water is sluggish and potentially dangerous if being chased.
2. **Global Threat Vector (3 floats)**:
   - `[64]`/`[65]`: Unit direction (east/north) to the nearest **alive** snake head.
   - `[66]`: Distance to that snake head, normalized as `min(dist / 150, 1)`. This gives the prey a global "danger sense" far beyond its local 8x8 vision, which is necessary since a snake can close an 8-cell gap very quickly.
3. **Reproduction Progress (1 float)**:
   - `[67]`: A scalar from `0.0` to `1.0` representing `grass_eaten / PREY_REPRODUCTION_GRASS`. This makes the reproduction goal directly observable.
4. **Grass Health Grid (64 floats)**:
   - `[68..132)`: An 8x8 grid showing the grass health (0.0 to 1.0) of the exact same cells as the entity grid. Used to guide the prey towards fresh pastures.

### Action Space (5 discrete actions)

Prey select from 5 possible moves each tick:
- `0`: Stand still
- `1`: Move Up (North)
- `2`: Move Right (East)
- `3`: Move Down (South)
- `4`: Move Left (West)

## Reproduction (Mitosis)

When a Prey accumulates enough consumed grass (`PREY_REPRODUCTION_GRASS = 50.0`), it undergoes reproduction (mitosis), splitting into two entities. This event gives a large terminal reward (`+25.0`) in training. 

To prevent spawn-camping exploits by predators, a Prey can only reproduce if there are no live snake heads within an 8-cell Chebyshev radius. When mitosis occurs, the original prey's episode terminates (as a success), and a new offspring is injected into the world nearby, inheriting the parent's `family_id`.

## Training Environment

Prey are trained via `learner/src/learner/train_prey.py` and the `RustPreyVecEnv` environment. They are trained in a mixed-model self-play environment:
- The environment spawns multiple Prey agents alongside frozen Snake models.
- The Prey slots are mixed: some slots actively learn and update the new policy, while others execute frozen historical checkpoints of the Prey model itself (e.g., using the `--existing` flag). 
- This shared-brain, mixed-generation approach ensures robust ecological dynamics without needing heavy multi-agent frameworks like PettingZoo.
